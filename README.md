# TopShelfRoles

TopShelfRoles is a Ruby on Rails application that aggregates and filters job postings into a unified dashboard, helping users quickly find fresh, relevant opportunities.

## Tech Stack

- Ruby on Rails
- PostgreSQL
- OpenAI API (planned)

## Planned Features

- Job ingestion from external sources
- Deduplication of listings
- Freshness filtering
- Dashboard view of recent roles
- AI-assisted role classification

## Status

Early development.
